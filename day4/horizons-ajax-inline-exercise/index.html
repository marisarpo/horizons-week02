Put the name of the city/metro area with the largest population in this <h1> tag

The goal of this exercise is to download data from a remote source using AJAX and then upload this page.

jQuery is laready loaded on this page. You should use it.

  1. Open http://horizons-json-cors.s3-website-us-east-1.amazonaws.com/data.json to check out what our data looks like.
  2. Load this data on this page using $.ajax().
  3. Find out which city is the largest by population.
  4. Display the name of the city in the h1 tag at the top, it has the id "out".
